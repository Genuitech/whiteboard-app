# COMC Operator One-Pager (Weekly Execution)

**Goal:** Run a no-shipping, no-grading, COMC-only card portfolio for consistent cashflow.

## 1) Core Limits (Always On)
- COMC only
- No grading, no physical shipping
- Max deployed capital: **85%**
- Cash reserve: **15% minimum**
- Max buy per card: **$125**
- Max exposure per player/team/IP/set: **15% of bankroll**
- Min estimated net margin after fees: **18%**

## 2) Buy Checklist (All Must Pass)
- [ ] At least 3 recent sold comps
- [ ] Current ask is 20%+ below normalized fair value
- [ ] Net margin estimate is 18%+
- [ ] Card price between $5 and $125
- [ ] No hype spike / unstable comp trend
- [ ] Duplicate count not excessive (max 3 same card unless high velocity)

## 3) Pricing Rules
- Initial list: **95%–102%** of normalized fair value
  - Fast-moving card: 95%–98%
  - Slower but quality spread: 99%–102%

### Markdown cadence
- Day 14: -4%
- Day 30: -6%
- Day 45: -8%
- Day 60: move to Exit Queue unless thesis still strong

## 4) Weekly Rhythm
### Monday
- Refresh comps
- Source candidates
- Enforce exposure and reserve limits

### Wednesday
- Reprice all active cards
- Apply markdown schedule
- Move stale cards toward Exit Queue

### Friday
- Run KPI review
- Calculate weekly + MTD net profit
- Reallocate budget to top-performing segments

## 5) Kill Switches (Pause New Buys)
Pause all new buys immediately if any trigger:
- 14-day sell-through < 8%
- 30-day realized net ROI < 3%
- Cash reserve < 10%
- Any category exposure > cap

### Recovery
- Stop buys 7 days
- Markdown stale inventory aggressively
- Rebuild reserve to 15%
- Resume only after metrics recover

## 6) KPI Snapshot (Track Weekly)
- Starting bankroll
- Deployed capital
- Reserve %
- New buys + cost
- Sales + gross proceeds
- Fees
- Net profit (week + MTD)
- Net ROI on deployed capital
- Sell-through (7/14/30 day)
- Aging buckets (0–30, 31–60, 61+)

## 7) Approval Required Only If
- Buy > $125
- Exception to buy rules
- Exposure cap override
- Deployment >85%

---
**Default setup:** $5,000 bankroll | $125 max buy | 18% margin floor | 15% reserve | 8% sell-through pause trigger.
